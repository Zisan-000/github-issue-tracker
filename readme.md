## What is the difference between var, let, and const?

Ans: var is function-scoped and allows redeclaration, let is block-scoped and allows reassignment, and const is block-scoped and cannot be reassigned.

## What is the spread operator (...)?

Ans: It is a syntax used to expand or unpack an iterable (like an array or an object) into individual elements or properties.

## What is the difference between map(), filter(), and forEach()?

Ans: map() creates a new array by transforming every element, filter() creates a new array with elements that pass a specific condition, and forEach() simply loops through elements without returning a new array.

## What is an arrow function?

Ans: It is a shorter, more concise syntax for writing JavaScript functions that also inherently inherits the this context from its surroundings.

## What are template literals?

Ans: They are strings enclosed in backticks (`) that allow for multi-line text and easy variable injection using the ${variable} syntax.
